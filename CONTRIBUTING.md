# Contributing to Think Local

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Open `Package.swift` in Xcode 26+
4. Create a branch for your changes

## Development

- **Build**: `swift build`
- **Test**: `swift test`
- **Run**: `swift run ThinkLocal` or press ⌘R in Xcode

## Guidelines

- Use SwiftUI and Apple frameworks only (no external dependencies)
- Follow Swift API Design Guidelines
- Write tests for new functionality
- Keep commits focused and descriptive

## Pull Requests

- One feature or fix per PR
- Include tests
- Update README if adding user-visible features
- PRs are squash-merged

## Code of Conduct

Be kind. Be constructive. We're all here to learn.
